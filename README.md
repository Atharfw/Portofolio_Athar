### Hello everyone this is My Personal Portfolio 

A minimalist, high-performance personal portfolio website built to showcase my projects, professional experience, and technical skills bridging the gap between Software Engineering and Cybersecurity.

### About The Project

This repository contains the source code for my personal portfolio. It is designed with a sleek, dark-themed UI to provide a clean user experience while highlighting my core competencies in web development, system architecture, and vulnerability assessment.

### Key Features
- **Project Showcase:** Detailed breakdowns of academic and personal projects with a custom Lightbox image gallery.
- **Interactive UI:** Smooth animations, hover states, and responsive design optimized for all devices.
- **Fast & Modern:** Built using the latest Next.js App Router for optimal performance and SEO.

### Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Library:** [React](https://reactjs.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Deployment:** Vercel

### Featured Projects Highlighted
A glimpse of the projects featured in this portfolio:
- **Offensive Security Final Project (KDJ):** A sanctioned penetration testing simulation focusing on critical vulnerabilities (SQLi, XSS) and comprehensive mitigation reporting.
- **UniTask:** A hyperlocal gig economy web platform tailored for student environments.
- **NongkiBib:** A gamified time-management mobile application concept enforcing strict scheduling penalties.
